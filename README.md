# expand

[![Build Status](https://travis-ci.org/DeidaraC/expand.svg?branch=master)](https://travis-ci.org/DeidaraC/expand)

the unix command `expand`, see `man expand`.

## Usage

expand [OPTION]... [FILE]...


## Support Options

* `-t`: how many space one tab(default is 8).
